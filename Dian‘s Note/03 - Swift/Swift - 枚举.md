## 枚举

OC语言的枚举 本质就是整形

先来看看怎么写的；

```swift
enum Num{
	case:	one
  case: two
	case: three
	case: four
}	
enum Num{
  case: one, two, three, four
}


var nu = Num.one
nu = Num.two
nu = .three
print(nu)

```
